CPU·코어·프로세스·스레드 관계

운영체제 관점에서 코어가 실제 명령을 실행하며, 코어가 직접 실행하는 단위는 스레드입니다. 프로세스는 하나 이상의 스레드를 포함하는 실행 중인 프로그램으로, 스레드를 통해 CPU에서 실행됩니다.

CPU
여러 코어를 담을 수 있는 칩. 스케줄링의 실질 대상은 스레드.
코어
명령을 실제로 실행하는 연산 단위. 한 시점에 1스레드 실행이 기본이며, 코어가 여러 개면 병렬 실행 가능.
프로세스
독립 주소 공간과 자원을 가진 실행 단위(파일/소켓 핸들, 보안 문맥 등 포함). 하나 이상의 스레드를 가짐.
스레드
코어가 직접 실행하는 최소 실행 단위. 같은 프로세스의 스레드끼리는 메모리 공유.

중요: 한 코어는 한 순간에 오직 하나의 스레드만 실행합니다. 여러 프로세스 실행은 컨텍스트 스위칭으로 교대로 이뤄지며, 멀티코어는 코어 수만큼 스레드를 동시에 처리합니다.

실행 관점 요약

  • 실행 주체: 코어 → 스레드
  • 프로세스 병행: 컨텍스트 스위칭으로 교대
  • 진짜 병렬: 멀티코어에서 코어 수만큼 병렬 스레드
